Bonus: Stock Exchange Quote Service App

Bonus: Stock Exchange Quote Service App

Let's build a little more practical Sinatra App - Stock Exchange Quote Service.

Our small Sinatra app allows us to retrieve different pieces of information about a particular stock.

For this app, we shall be using the open-uri library. The open-uri library extends Kernel#open and helps us in accessing the web. It accepts URIs for FTP and HTTP as well as local filenames. Once opened, these resources can be treated as if they were local files, accessed using conventional IO methods. The URI passed to open is either a string containing an HTTP or FTP URL. When opening an HTTP resource, the method automatically handles redirection and proxies. The second library is csv, and it is capable of parsing comma-separated value files.

We shall be using Yahoo! Finance to pull the information of a stock. The url we are going to use is –
http://download.finance.yahoo.com/d/quotes.csv?s=#{stock_symbol}&f=sl1d1t1c1ohgv&e=.csv

For example, I used – http://download.finance.yahoo.com/d/quotes.csv?s=bcs&f=sl1d1t1c1ohgv&e=.csv

and downloaded the file quotes.csv whose contents were – 

"BCS",8.26,"3/30/2009","4:00pm",-1.63,8.73,8.75,8.05,11491272

Note that in case the url changes, you must find the new one.
